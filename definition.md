# Note

A minimal text editor. Black and white. Borderless.

It opens files from the operating system — `.md`, `.py`, `.txt` — and lets you edit them as plain text. Nothing is rendered. Nothing is highlighted. You see the words as they are.

The window has no frame. A thin white border, rounded at the corners, holds a black rectangle. At the top, a File button on the left, a close button on the right. Below, a text area. That is all.

When you open a file, its name appears in the title. When you change something, a star appears beside the name. When you close, it saves without asking. If there is nowhere to save, it asks once.

Ctrl+O opens. Ctrl+S saves. The caret blinks. The rest is typing.

Note is not an IDE. It is not a markdown viewer. It is not a file manager. It has no tabs, no plugins, no settings, no themes, no preferences. It does one thing: it puts words in a file.

That is Note.
